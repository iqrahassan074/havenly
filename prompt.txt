Tech Stack:
- Next.js (latest version)
- React
- Tailwind CSS
- Framer Motion (for animations)
- Optional: React Icons

- Full-width hero with elegant furniture image or illustration

4️⃣ Testimonials Section

6️⃣ Navbar & Footer
- Responsive sticky navbar with smooth scroll highlight  
- Hamburger menu animation on mobile  
- Footer with social icons and links, hover animations  

7️⃣ UI/UX Features
- Dark mode toggle  
- Smooth scroll for internal links  
- Modern, clean color palette (light, neutral, soft accents)  
- Typography hierarchy: headings, body, buttons  
- Micro-interactions: hover effects on buttons, links, product cards  

8️⃣ Project Structure
- Reusable React components

9️⃣ Output
- Fully interactive frontend code
